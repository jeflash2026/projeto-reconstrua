# Projeto Reconstrua

Produto novo e totalmente independente. Não compartilha código, arquitetura, configuração ou estrutura com nenhum projeto anterior.

## Fonte de verdade

A única fonte oficial de verdade é o **LIVRO MESTRE** (o Canon), em [`constitution/`](constitution/) — cinco volumes: 00 Constituição · 01 Ontologia · 02 Epistemologia · 03 Regras Operacionais · 04 Governança. Índice: [constitution/INDEX.md](constitution/INDEX.md).

- Nenhuma linha de código pode existir sem origem no Canon.
- Toda implementação cita explicitamente o capítulo/entidade do Canon que a origina.
- Em caso de dúvida, o Canon sempre prevalece; toda evolução futura obedece ao Volume 04 (Governança).
